# ToDoListWebApp

My first project in Java

A simple todo lists web app built using Jakarta EE

## Jakarta EE technologies used:
* Servlets
* JSP
* JDBC

## Application Server:
* Apache Tomcat

## Tool Build Project:
* Apache Maven

## Other technologies used
* Javascript (inside the JSP file)
* Tailwind (using CDN)
* MySQL
